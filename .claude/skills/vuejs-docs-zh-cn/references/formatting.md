# 文本格式

## 译注写法

在需要注释的位置添加角标：

```html
... <sup>[[1]](#footnote-1)</sup> ...
```

文章末尾加入译注内容：

```html
<small>
  __译者注__
  <a id="footnote-1"></a>[1] ... <a id="footnote-2"></a>[2] ...
</small>
```

## 标点符号

| 符号类型                                 | 规则                   |
| ---------------------------------------- | ---------------------- |
| 逗号、句号、分号、冒号、叹号、问号、顿号 | 全角：`，。；：！？、` |
| 引号                                     | 全角：`""` 和 `''`     |
| 括号                                     | 半角：`()`             |
| 破折号                                   | `——`                   |
| 省略号                                   | `……`                   |
| 间隔号                                   | `·`                    |
| 书名号                                   | `《》` 和 `〈〉`       |
| 分隔号                                   | `/`                    |

> 代码部分保留英文标点。

## 内联代码

- 务必用反引号括起代码或代码关键字。
- 代码注释中的代码/关键字也要用反引号括起。

## 空格规则

- 两个英文单词之间一个空格：`Hello World`
- 中英文之间一个空格：`中文 English 中文`
- 全角标点两边无空格：`中文，中文。`
- 半角括号内侧无空格，外侧与中英文之间有一个空格：`中文 (中文) 中文`
- Markdown 标记（加粗、斜体、链接）外侧的空格应保证渲染后符合上述规则：
  ```
  遵守 **JavaScript 编码规范**非常重要
  遵守 [JavaScript 编码规范](#foo)非常重要
  ```

## 加粗与斜体格式

1. 中文翻译统一使用星号标注加粗和斜体，不用下划线。
2. 无法正确渲染时可用 `<strong>` 或 `<em>` HTML 标记替代。
3. 早期文档中通过调整标点解决渲染问题的做法可酌情保留。
